DogChain Development Details

Source Code

The source code for DogChain is located in the Github repository for our team, TeamOfPeople. To find the software for our final deliverable, you must access the Alpha files. Within the Alpha files, there are four folders. Their contents are described as follows.

  • Scenes: the two scenes used in our game (lobby and game-play scenes)
  • Scripts: the C# scripts used to control the game
  • Textures: the artistic assets used in the game
  • audio: the recordings used as audio for the game
  • Visit Repo
Path to Alpha Files: DogChain/Assets/Alpha files/
